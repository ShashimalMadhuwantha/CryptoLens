# CryptoLens - Crypto Market Tracker

CryptoLens is a web application that provides real-time cryptocurrency market data, trends, and news. Built with modern web technologies, CryptoLens offers a user-friendly interface for crypto enthusiasts to stay updated with the latest market insights.

---

## Features

1. **Real-Time Crypto Market Data**
   - Track live prices and market movements.
   - Powered by the CoinGecko API.

2. **Top Cryptocurrencies Section**
   - Displays the top-performing cryptocurrencies by market capitalization.

3. **Detailed Insights**
   - Includes price, 24-hour change, market cap, and trading volume for each cryptocurrency.

4. **Search Functionality**
   - Search and view details for specific cryptocurrencies.

5. **Interactive Charts**
   - Visualize price trends with Chart.js.

6. **Crypto News**
   - Stay updated with the latest crypto news, fetched from the Crypto News API.

7. **Responsive Design**
   - Mobile-first design with Bootstrap 5 for seamless access across devices.

---

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Frameworks/Libraries**: 
  - Bootstrap 5 (Responsive Design)
  - Chart.js (Interactive Charts)
- **APIs**: 
  - [CoinGecko API](https://www.coingecko.com/en/api) for real-time market data
  - [Crypto News API](https://cryptonews-api.com/) for news updates
